# Deferred Roadmap

Отложенные идеи проекта «Бухта».

Статус: `Draft`.

Правило: сюда попадают только осознанно отложенные идеи, которые не являются активным планом и не являются багом.

## Production History

- Расширить вкладку `История` заведующего производством до рабочего журнала операций: приход сырья, приход тары, выпуск продукции, передача на распределитель.
- Добавить фильтр по типу операции и сортировку по дате.
- Рассмотреть поиск по продукции, сырью, таре или распределителю после появления достаточного объема операций.

## Reports And Control Pack

- Сформировать следующий управленческий reports/control contour поверх уже реализованных read models.
- Добавить export/print для управленческой истории и отчетов.
- Добавить аналитику скидок, товаров, клиентов и cash movement.
- Рассмотреть сохранение периода/фильтров для контрольных экранов.
- Добавить отдельный коммерческий срез без доступа к директорским данным.

## Desktop And Tablet Expansion

- Оставить mobile shell основным для ежедневных операций.
- Для history/control/admin списков добавить tablet/desktop layout с широкой таблицей, закрепленными фильтрами и лучшим использованием горизонтального пространства.
- Приоритетные кандидаты: история операций, история продаж, аналитика, пользователи, клиенты и каталог.
